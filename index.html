    그럼 이제는 이걸 실제 데이터로 저장해서 앱을 껏다 키더라도 적용될 수 있도록 해보자.

     

    일단 당작은 데이터의 규모가 크지 않으므로 shared_preferences를 사용해보려고 한다.

     

    https://pub.dev/packages/shared_preferences

     

    Cursor의 프롬프트와 공식 라이브러리 페이지를 참고해서 코드를 짜보도록 하자.

    home/word_model.dart 
    		class Word {
    		  final String word;
    		  final String meaning;
    		  final String date;
    		 
    		  Word({required this.word, required this.meaning, required this.date});
    		 
    		  factory Word.fromJson(Map<String, dynamic> json) {
    		    return Word(
    		      word: json['word'],
    		      meaning: json['meaning'],
    		      date: json['date'],
    		    );
    		  }
    		}

     

     

    home/screen.dart 
    		import 'dart:convert';
    		 
    		import 'package:flutter/cupertino.dart';
    		import 'package:flutter/material.dart';
    		import 'package:get/get.dart';
    		import 'package:quest/home/item.dart';
    		import 'package:quest/home/word_model.dart';
    		import 'package:shared_preferences/shared_preferences.dart';
    		 
    		class HomeScreen extends StatefulWidget {
    		  const HomeScreen({super.key});
    		 
    		  @override
    		  State<HomeScreen> createState() => _HomeScreenState();
    		}
    		 
    		class _HomeScreenState extends State<HomeScreen> {
    		  List<Word> _words = [];
    		 
    		  late SharedPreferences prefs;
    		 
    		  Future<void> _loadWords() async {
    		    prefs = await SharedPreferences.getInstance();
    		    final words = prefs.getStringList('words') ?? [];
    		 
    		    _words = words.map((e) => Word.fromJson(jsonDecode(e))).toList();
    		 
    		    setState(() {});
    		  }
    		 
    		  Future<void> _handleDelete(Word item) async {
    		    final words = prefs.getStringList('words')?.toList() ?? [];
    		 
    		    for (var word in words) {
    		      final json = jsonDecode(word);
    		      if (json['word'] == item.word) {
    		        words.remove(word);
    		        break;
    		      }
    		    }
    		 
    		    prefs.setStringList('words', words);
    		  }
    		 
    		  @override
    		  void initState() {
    		    super.initState();
    		    _loadWords();
    		  }
    		 
    		  @override
    		  Widget build(BuildContext context) {
    		    return CupertinoPageScaffold(
    		      // navigationBar: const CupertinoNavigationBar(
    		      //   middle: Text('홈'),
    		      // ),
    		      child: SafeArea(
    		        child: Stack(
    		          children: [
    		            ListView.separated(
    		              itemCount: _words.length,
    		              itemBuilder: (context, index) {
    		                final Word item = _words[index];
    		 
    		                return WordItem(
    		                  item: item,
    		                  handleDelete: (item) {
    		                    _handleDelete(item);
    		                    _loadWords();
    		                  },
    		                );
    		              },
    		              separatorBuilder: (context, index) {
    		                return const Column(
    		                  children: [
    		                    Divider(
    		                      color: Colors.black12,
    		                      height: 1,
    		                    ),
    		                  ],
    		                );
    		              },
    		            ),
    		            Positioned(
    		              bottom: 0,
    		              right: 20,
    		              child: GestureDetector(
    		                onTap: () async {
    		                  await Get.toNamed('/word');
    		                  _loadWords();
    		                },
    		                child: Container(
    		                  width: 60,
    		                  height: 60,
    		                  decoration: BoxDecoration(
    		                    color: Colors.orange.shade100,
    		                    borderRadius: BorderRadius.circular(100),
    		                    border: Border.all(
    		                      color: Colors.orange.shade200,
    		                      width: 1,
    		                    ),
    		                  ),
    		                  child: const Icon(
    		                    CupertinoIcons.plus,
    		                    color: Colors.orangeAccent,
    		                  ),
    		                ),
    		              ),
    		            ),
    		          ],
    		        ),
    		      ),
    		    );
    		  }
    		}

    일단 이번에 알게 된 점은,

    Flutter에서는 late라는걸 사용해서 나중에 선언될 변수를 작성할 수 있고,

    Future를 통해 리액트의 Promise 처럼 사용 가능한 것 같다.

     

    SharedPreference는 getInstance를 통해 인스턴스를 가져와서 사용하며,

    LocalStorage처럼 사용할 수 있는 공간인 것 같다.

    구현에 집중하느라 딥하게 사용해보지는 않았는데,

    이 부분은 집중해서 사용해볼 필요가 있을까...?

     

    싶었다가 어차피 모바일기기에서는 SQLite를 써도 되니까 그냥 넘어가기로 했다
